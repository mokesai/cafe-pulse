<task type="auto">
  <name>Task 1: Add tenant-flag support to sync-square-catalog.js</name>
  <files>scripts/sync-square-catalog.js</files>
  <action>
Add tenant-aware credential loading to `scripts/sync-square-catalog.js`. This is the template pattern that the other two scripts will follow.

1. **Add tenant flag parsing to `parseArgs()` (or equivalent):**
   ```javascript
   // After existing arg parsing
   let tenantId = null
   let tenantSlug = null
   const tenantIdArg = args.find(arg => arg.startsWith('--tenant-id='))
   if (tenantIdArg) tenantId = tenantIdArg.split('=')[1]
   const tenantSlugArg = args.find(arg => arg.startsWith('--tenant-slug='))
   if (tenantSlugArg) tenantSlug = tenantSlugArg.split('=')[1]
   ```

2. **Add slug-to-ID resolution function:**
   ```javascript
   async function resolveTenantBySlug(supabase, slug) {
     const { data, error } = await supabase
       .from('tenants')
       .select('id')
       .eq('slug', slug)
       .eq('is_active', true)
       .single()
     if (error || !data) throw new Error(`Tenant not found for slug: ${slug}`)
     return data.id
   }
   ```

3. **Add Vault credential loading function:**
   ```javascript
   async function loadTenantSquareCredentials(supabase, tenantId) {
     const { data, error } = await supabase.rpc('get_tenant_square_credentials_internal', {
       p_tenant_id: tenantId
     })
     if (error || !data || data.length === 0) {
       throw new Error(`Failed to load Square credentials for tenant ${tenantId}: ${error?.message || 'No data returned'}`)
     }
     return data[0]
   }
   ```

4. **Update the main function to resolve tenant and load credentials:**
   At the start of the main execution flow:
   ```javascript
   // Resolve tenant
   if (tenantSlug) {
     tenantId = await resolveTenantBySlug(supabase, tenantSlug)
     console.log(`Resolved tenant slug "${tenantSlug}" to ID: ${tenantId}`)
   }

   let squareAccessToken, squareEnvironment, squareLocationId
   if (tenantId) {
     // Load from Vault
     const creds = await loadTenantSquareCredentials(supabase, tenantId)
     squareAccessToken = creds.access_token
     squareEnvironment = creds.environment
     squareLocationId = creds.location_id
     console.log(`Loaded Square credentials for tenant ${tenantId} from Vault`)
   } else {
     // Default: use env vars (backward compatible)
     squareAccessToken = process.env.SQUARE_ACCESS_TOKEN
     squareEnvironment = process.env.SQUARE_ENVIRONMENT
     squareLocationId = process.env.SQUARE_LOCATION_ID
   }
   ```

5. **Move the module-level Square env var reads into the main function:**
   Remove the top-level:
   ```javascript
   const squareAccessToken = process.env.SQUARE_ACCESS_TOKEN
   const squareEnvironment = process.env.SQUARE_ENVIRONMENT
   const squareLocationId = process.env.SQUARE_LOCATION_ID
   ```
   These are now resolved conditionally inside main().

6. **Update the SQUARE_BASE_URL derivation** to use the resolved `squareEnvironment` variable instead of the module-level constant.

7. **Update `showUsage()` to document the new flags:**
   Add:
   ```
   --tenant-id=UUID    Target a specific tenant by UUID
   --tenant-slug=SLUG  Target a specific tenant by slug (resolved to UUID)
   ```

Keep all existing sync logic unchanged. Only the credential source changes.
